# Resort Environment Prompt Specifications

Every asset inherits the twenty-field contract and shared visual/negative blocks in ../../07-image-prompts.md. Empty plates contain no person. The resort is fictional/generic enough to avoid implying a real property partnership, yet structurally plausible and reproducible.

## Shared fields 2–20

2. Inputs: owner-approved architecture/material references controlling only named qualities; no hotel identity, logo, signage, or copyrighted signature décor.
3. Identity: no person in empty plates; later character assets use CHR-SAGE-CONT-001.
4. Scene: premium tropical coastal resort with a walkable path, restrained architecture, water/horizon, selective foliage.
5. Placement/scale: human-scale path/openings/railings/steps; fixed route and settle mark.
6. Performance: no people/crowds; foliage/water movement only in later motion tests.
7. Materials: warm plaster, pale stone, restrained timber, coherent paving, sand, water, selected coastal planting.
8. Camera: rectilinear level lens; matching perceived height through KF-04; no drone/fisheye/impossible crane.
9. Output: named diagnostic 16:9 or 9:16 target; safe zones documented outside pixels.
10. Geometry: fixed horizon, waterline, path, building edge, major palm, pool/beach boundary, and sun direction.
11. Light: warm plausible daylight with coherent shadows/reflections and highlight protection.
12. Palette: ivory, warm sand/stone, controlled blue-green water/sky, natural foliage; no orange-teal cliché.
13. Layers: path/foreground, subject route/architecture middle, water/horizon/background, selective foliage framing.
14. Continuity: ENV-RESORT-001 is canonical; all derivatives preserve anchors and screen direction.
15. Preserve: structure, route, horizon, sun, materials, anchor objects, hospitality readability.
16. Prevent: logos/text/watermarks, named-property cues, impossible architecture/water, changing palms, decorative clutter, duplicate objects.
17. Output: versioned lossless review asset with hash/prompt/settings; later approved optimized derivative only.
18. Downstream: keyframes, motion, final poster, deterministic web media.
19. Review: structural plausibility, anchor overlay, material/light consistency, safe zones, claim/brand clearance.
20. Provenance/rights/state: PROPOSED, private, owner-reviewed before use; reference rights logged.
